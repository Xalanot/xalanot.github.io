# C string
In C++ the are two different types of strings. The old <strong>c string</strong> and the <strong>std::string</strong>. In this
post I will talk about the c string and in the upcomming post I will continue with std::string. 

### Introduction
A c string is nothing more than an array of chars. But to be a valid c string a terminating <strong>null character</strong> is
needed. This null character is the ASCII character with a value of 0.
